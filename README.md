The aims of this repository are to refresh my knowledge of the Python language and explore TUI applications.

The database consists of files divided by year in the following format:
- Films
    Dave made a maze/Labirynt Dave'a (2017) **Bill Watterson**
    Gwiezdne wojny: Powrót Jedi VI (1983) **Richard Marquand**
- Books
    Uczeń Diabła **Kenneth B. Andersen**
    Czterdzieści i cztery **Krzysztof Piskorski ebook**
- The file names are in format 'Films YEAR' and 'Books YEAR'
- Create a subwindow with a border around it.
- At the top add a Films and Books selector.
- Allow switching between them using the 'h' & 'l' keys and left and right arrow keys.
- Display all entries in one pad seperated by years.
- Allow scrolling through entries using the 'j' & 'k' keys and up and down arrow keys.
- Allow scrolling through entries using the 'd' & 'u'.
- If an entry is in the wrong format display a red background.

- Add an filter, activated with a 'f' key with fuzzy finder.
- If an entry appears more than once, display its order number on the right side of the entry.
- Add scroll indicator.
